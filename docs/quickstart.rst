Example script

The following script shows how to use this driver on a Windows system that is attached to COM1 serial port

.. literalinclude:: ../examples/example_uart_svm41.py

After installing all required packages the you can start it on the command line by typing.

.. sourcecode:: bash

    python ../examples/example_serial_svm41.py --port <your COM port>
